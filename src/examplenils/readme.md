# Example Exocortex: Nils Pihl, CEO of Auki Labs

This is a real, filled-in exocortex. Use it to get a sense of what each file does and how they work together before setting up your own.

## What's in here

| File | What it does |
|------|-------------|
| **claude.md** | Instructions for the AI agent — the reading order and alignment checks it follows every session |
| **identity.md** | Who the owner is, how they see the world, their public bio |
| **values.md** | Personal guiding values with explanations of what each one means in practice |
| **organization.md** | The company's mission, context, and cultural values |
| **role.md** | The owner's responsibilities and recurring routines as CEO |
| **methods.md** | Mental models and frameworks the owner uses regularly (NASA, LEIA, Why/How/What) |
| **glossary.md** | Domain-specific terms the agent needs to use correctly |

## How to use this

Read through the files to see the level of detail and the kind of content that makes an exocortex useful. Then use the `exocortexTemplate/` folder to build your own — it has the same structure with guiding prompts instead of Nils's specific content.

The most important thing is that the content is *yours*. An exocortex only works if it reflects how you actually think, what you actually do, and what you actually care about. Copy the structure, not the answers.
