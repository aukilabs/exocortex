# STYLE

## Summary

- Lead with the answer.
- Prefer short paragraphs over long preambles.
- Use bullets only when structure helps.
- Be specific, concrete, and low-hype.
- Separate decision, risk, and next step when clarity matters.

## Writing Defaults

- Internal updates: terse and operational.
- External writing: precise, readable, and calm.
- Documentation: summary first, details below.
